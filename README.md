## Sobre o projeto  
O **Smart Collect** é uma solução tecnológica inovadora para o gerenciamento inteligente de resíduos sólidos urbanos. Combinando **IoT, sensores em tempo real e algoritmos inteligentes**, o sistema otimiza a coleta seletiva, promove a sustentabilidade e reduz custos operacionais.  

O objetivo é apoiar cidades e comunidades no desenvolvimento de práticas mais **sustentáveis, econômicas e eficientes**, alinhadas aos princípios de **Smart Cities**.  

O projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** no curso de **Análise e Desenvolvimento de Sistemas** pela **FATEC Zona Leste**.  

---

##  Funcionalidades  
-  **Monitoramento em tempo real** dos níveis de resíduos nos coletores.  
-  **Dashboard interativo** para análise de dados e relatórios de desempenho.  
-  **Notificações inteligentes** para coleta no momento ideal.  
-  **Incentivo à sustentabilidade**, promovendo práticas ecológicas.  
-  **Suporte a cidades inteligentes**, com base em IoT e análise de dados.  

---

##  Tecnologias utilizadas  

O **Smart Collect** foi desenvolvido com um conjunto de tecnologias modernas que garantem desempenho, escalabilidade e integração entre hardware e software. No front-end utiliza-se **HTML, CSS, JavaScript e React.js** para construir interfaces intuitivas e responsivas, enquanto no back-end a aplicação é sustentada por **Node.js** e **C++**, oferecendo robustez e processamento eficiente. O banco de dados é implementado com **Firebase**, garantindo armazenamento seguro e em tempo real. Para a camada de IoT, o sistema emprega **sensores ultrassônicos** conectados ao **ESP32**, responsáveis por captar os dados dos coletores. Por fim, a infraestrutura conta com **Docker/Podman** para containerização e **APIs REST**, assegurando integração e comunicação ágil entre os módulos.
